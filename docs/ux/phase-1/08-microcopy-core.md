# 08: コアマイクロコピー
_Core Microcopy Specification_

---

## 概要

アプリ内で使用するコアマイクロコピー（空状態、エラー、記録完了）を定義する。
Voice & Toneガイドに基づき、一貫した言葉づかいを実現する。

**関連イシュー:** UX-060, UX-061, UX-062, UX-063

---

## Voice & Tone

### Voice（声）— 5つの形容詞

1. **静か** — 情報量が少ない、余白がある
2. **やさしい** — 判断しない、責めない
3. **軽い** — 短い、押し付けない
4. **誠実** — 嘘をつかない、過剰に盛らない
5. **ほのかにワクワク** — 儀式の締めだけ少し明るい

### Tone（調子）— 5原則

1. **急かさない** — 「今すぐ」「早く」は使わない
2. **褒めすぎない** — 「最高！天才！」は避ける
3. **意味づけしない** — 人生論を語らない
4. **視線は"横"** — 上から指導しない、下から媚びない
5. **具体的で短い** — 情緒語は控えめ

### 敬語レベル

- 基本: **丁寧語（です・ます）**
- 硬すぎない短文中心
- タメ口・敬語過多は禁止

## 推奨語彙 / 禁止語彙

### 望ましい語彙

```
静かに、そっと、今日、ひとつ、積み重なり、
追加、記録、余韻、片付け、残す、増える
```

### 避けたい語彙

```
今すぐ、早く、急いで、最高、天才、神、
成功、勝ち、負け、変われる、成長できる、
がんばれ、完璧、すごい、やったね
```

## 空状態メッセージ

### 棚図鑑（カードなし）

```
まだ何もありません。

最初のひと皿を、
ここに残してみませんか。
```

**NG例**: 「棚が空っぽです！今すぐ記録しよう！」

### 検索結果なし

```
見つかりませんでした。

別の言葉で探してみてください。
```

**NG例**: 「残念！該当する料理がありません」

### 写真なし（カード詳細）

```
写真はまだありません。

次に作るときに、追加できます。
```

**NG例**: 「写真がないと寂しいです！」

## エラーメッセージ

### 原則

- 責めない
- 急かさない
- 説明しすぎない
- 起きたことだけを静かに伝える

### 保存失敗

```
保存できませんでした。

もう一度お試しください。
```

### 読み込み失敗

```
読み込めませんでした。

少し待ってから、
もう一度お試しください。
```

### 写真アクセス失敗

```
写真にアクセスできませんでした。

設定から許可をご確認ください。
```

### ネットワークエラー（将来用）

```
接続できませんでした。

オフラインでも記録は残ります。
```

## 記録完了メッセージ

### バリエーション（ランダム表示）

1. ```記録しました。```
2. ```棚に1枚、追加されました。```
3. ```今日の分、残しておきます。```
4. ```また作った日が増えました。```
5. ```静かに、積み重なりました。```
6. ```今日のひと皿、残しました。```
7. ```棚が少し育ちました。```

### 表示ルール

- ランダムで選択
- 連続で同じメッセージは避ける
- セレブレーションアニメーションと同時に表示

### NG例

```
やったね！最高！
完璧な記録です！
すごい！継続できてますね！
天才シェフ！
```

## グレードアップ時（追加メッセージ）

```
{料理名}が育ちました。
```

シンプルに。「レベルアップ」等のゲーム用語は禁止。

## 「AIっぽくない」チェックリスト

コピーを書いたら、以下を確認：

- [ ] 過度に丁寧すぎないか
- [ ] 「！」を多用していないか
- [ ] 定型文的な表現になっていないか
- [ ] 評価・判断を含んでいないか
- [ ] ユーザーを煽っていないか
- [ ] 長すぎないか（1-2文が理想）
- [ ] 感情を押し付けていないか
- [ ] 「素晴らしい」「素敵」等を使っていないか
- [ ] 命令形になっていないか
- [ ] 比較を促していないか
