### Kitchen Signals 入力仕様（UX-015）

#### 入力項目一覧

| 項目 | 選択肢 | デフォルト | 必須 |
|------|--------|-----------|------|
| 年代 | 20-30代 / 40-50代 / 60代〜 | 未選択 | × |
| 世帯 | ひとり暮らし / ファミリー | 未選択 | × |

**注:** 「好きな料理ジャンル」は MVP では含めない。理由:
- 入力負荷を最小化
- 行動シグナルで自然に蓄積（World Bible準拠）
- Phase 2 で提案機能と共に追加検討

#### 入力UI形式

- **単一選択ボタン（Chip/Pill）**
- タップで選択、再タップで解除
- 複数項目の同時選択なし（年代・世帯それぞれ1つ）
- 未選択のまま「次へ」可能

#### 「分類されている感」回避チェックリスト

- [x] 「あなたのタイプは〜」と言わない
- [x] 選択結果を画面に大きく表示しない
- [x] 選択で画面が劇的に変わらない
- [x] いつでも変更できることを明記
- [x] 選択肢に優劣をつけない
