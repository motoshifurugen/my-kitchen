# わたしの台所図鑑 (Watashi no Daidokoro Zukan)

> 料理は静かな自己回復の儀式である

## 概要

感覚的な料理ジャーナルアプリ。
レシピ管理や在庫追跡ではなく、料理を個人的な儀式として記録する体験駆動型アーカイブ。

## 特徴

- **3タップ記録**: レシピ選択 + 写真 + メモ（日付・調理回数は自動記録）
- **感情ルート**: 料理体験の「気分」を選択し、UIの雰囲気に反映
- **ライティング温度**: 時間帯とルートに基づくダイナミックUIテーマ
- **長期的アーカイブ**: 個人的な「台所の記憶」を蓄積

## 技術スタック

- **iOS**: Swift
- **Android**: Kotlin
- **設計思想**: ネイティブ開発、クロスプラットフォーム抽象化なし
